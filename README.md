# 🐱 고양이 장애물 피하기 게임

Python으로 만든 리얼타임 CLI 게임입니다. 자동으로 내려오는 장애물을 피해 생존하세요!

## 🎮 게임 설명

- **목표**: 15개의 장애물을 피해 살아남기
- **조작**: `a`(왼쪽), `d`(오른쪽), `q`(종료)
- **난이도**: 매우 높음 (12x8 좁은 보드, 다량의 장애물)

## 📋 필수 요구사항

- Python 3.6 이상

## 🚀 실행 방법

### 1. 코드 다운로드
```bash
git clone https://github.com/your-username/cat-dodge-game.git
cd cat-dodge-game
```

### 2. 게임 실행
```bash
python3 cat_dodge_game.py
```

## 🎯 게임 규칙

1. 🐱 고양이가 화면 최하단에 있습니다
2. 위에서 `#` 장애물이 자동으로 내려옵니다
3. 한 번에 여러 장애물이 동시에 나타날 수 있습니다
4. 15개를 모두 피하면 **승리! 🏆**
5. 장애물과 부딪히면 **게임오버! 💥**

## ⌨️ 조작법

| 입력 | 동작 |
|------|------|
| `a` | 왼쪽으로 이동 |
| `d` | 오른쪽으로 이동 |
| `q` | 게임 종료 |

## 🎨 게임 설정 (난이도 조절)

`cat_dodge_game.py`의 `__init__` 메서드에서 다음을 조정할 수 있습니다:

```python
def __init__(self, width=12, height=8):
    # width: 보드 가로 크기 (12 = 좁음, 20 = 넓음)
    # height: 보드 세로 크기 (8 = 짧음, 15 = 김)
    self.target_score = 15  # 피해야 할 장애물 개수
```

`game_loop()`의 `time.sleep()` 값을 조정하면 속도를 변경할 수 있습니다:

```python
time.sleep(0.6)  # 값이 작을수록 빠름
```
